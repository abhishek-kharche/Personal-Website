Projects..

  • Android Programming - Independent Study

    • An ongoing project based on Android Programming under the supervision of Prof. Stephan Bohacek. The objective of project was to determine user's physical location based of accelerometer and Longitude/Latitude values. Finally trace an itinerary for user's day activity. [GitHub]

  • Information Retrieval for Text Analysis

    • This is the project to implement Information Retrieval for Text Analysis. I used open source Java Software jEdit4.3 for analyzing the text content inside methods. The project extracts the method contents from the source and create a corpus of terms after splitting and stemming the words. This corpus is then parsed and similarity against user queries for each document is found. These results are then compared against the GoldSets which are the sets evaluated by someone in past years.
    Scripting Language:- Perl, Java, Shell scripting. [GitHub]

  • Session Track - Information Retrieval

    • This project was completed in Spring '13 for Information Retrieval course with a team strength of 2 people. Project aim was to compare the retrieval runs of Session track in TREC-8 systems with the runs submitted by participants in past years. Detailed information could be found at TREC. [GitHub]

  • Reverse Engineering - Software Design

    • This was the individual project to reverse engineer an open source software JavaPEG. The objective of a project was to obtain UML and other design diagrams from source code to assist developers. An extension was provided to the existing software for better GUI using Java Swings and AWT packages. [GitHub]

  • Book My Ticket

    • A project to design & build a website for travel agency. Customers can register themselves on a website, select several packages for tours & book them. Provision was made to modify the selection at checkout, password recovery was implemented. MS access was used in the back-end to construct the Database and front-end was programmed in HTML, JSP, AJAX and JavaScripts. [GitHub]

  • Character Writing Robot

    • A final year project based on 'Robotics' which allows the robot to write characters on page by holding a pen using 'Rack and Pinion' arrangement. The project was sponsored by and completed in collaboration with Indian Institute of Technology (IIT) - Bombay in ERTS lab.
